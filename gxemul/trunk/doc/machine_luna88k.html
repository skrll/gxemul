GXemul: Luna 88K emulation

Back to the index.

This emulation mode attempts to emulate a Luna 88K workstation. Note that these were based on Motorola 88000 processors, not 68000.

NOTE: The Luna 88K emulation mode is not fully working yet. OpenBSD/luna88k ramdisk kernels can reach userland and be interacted with, in serial console mode. Using a graphical framebuffer works but no keyboard is emulated yet, so it is not useful.

These instructions are here mostly for me to remember how to debug/proceed:

Download a RAMDISK kernel:

  • https://ftp.eu.openbsd.org/pub/OpenBSD/6.8/luna88k/bsd.rd

Start the emulator like this: 

	gxemul -e luna-88k bsd.rd

	(For older OpenBSD/luna88k kernels, 5.4 and below:)
	gxemul -e luna-88k 0x20000:0x20:bsd.rd

(Add -X to try the graphical framebuffer.)

NOTE:: As neither SCSI nor network is emulated yet, there is not much that can be done yet with the emulated machine, other than poke around in the ramdisk environment.

The following is not needed in order to experiment with booting OpenBSD, but can be interesting nonetheless: Download https://ftp.eu.openbsd.org/pub/OpenBSD/6.2/luna88k/boot and try 

	gxemul -v -e luna-88k 0x00700000:0x20:0x00700000:boot
